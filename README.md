# Hotel Reservation Optimizer

Mevcut otel rezervasyonlarini daha avantajli fiyatlarla karsilastirip guncellemeye odaklanan Python projesi.

Proje, daha once bireysel veya grup olarak rezervasyon yapmis musteriler icin konaklama tarihinden once ayni otelde daha uygun bir firsat olusup olusmadigini kontrol eder. Daha iyi bir fiyat bulunursa rezervasyon verisini avantajli sartlarla gunceller.

## Nasil Calisir?

- `reservations.csv` dosyasindaki mevcut rezervasyonlar okunur.
- `BugBusters.py` sunucudan guncel fiyat bilgisini ceker.
- Mevcut rezervasyonlar ile guncel fiyatlar karsilastirilir.
- Daha avantajli sonuc varsa `new_reservations2.csv` dosyasina guncellenmis kayitlar yazilir.

## Dosyalar

| Dosya | Aciklama |
| --- | --- |
| `reservations.csv` | Mevcut rezervasyon kayitlari |
| `BugBusters.py` | Fiyat verisini ceken ve karsilastirma yapan ana script |
| `new_reservations2.csv` | Guncellenmis rezervasyon ciktilari |
